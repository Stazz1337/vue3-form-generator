# FormGenerator - Генератор форм на Vue 3

Проект представляет собой универсальный генератор форм с использованием Vue 3, TypeScript, Vuex, VueRouter, VeeValidate, Yup, Vite. Включает в себя готовые страницы для регистрации, профиля, обратной связи и контактов с возможностью валидации данных.

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите проект в режиме разработки:

```bash
npm run dev
```

3. Соберите проект для продакшена:

```bash
npm run build
```

4. Запустите локальный сервер для проверки сборки:

```bash
npm run preview
```

5. Проверьте и исправьте код с помощью линтера:

```bash
npm run lint
```
